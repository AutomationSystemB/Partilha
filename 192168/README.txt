192168 é um programa em C# para definir o IP de uma qualquer interface de rede de um PC.

Apenas é preciso dar permissões à aplicação da primeira vez que esta é aberta, a partir daí é possivel trocar de IP as vezes que quiser, nas interfaces que quiser.

Para definir máscaras de rede diferentes de 255.255.255.0(por defeito) pode-se usar notação CIDR (ex: 192.168.1.2/16 para usar máscara 255.255.0.0)
